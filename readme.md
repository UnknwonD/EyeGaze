# Eye Tracking AI with Detectron2 and YOLOv5

![Eye Tracking Banner](Logo.png)

## 📖 Introduction

이 프로젝트는 Detectron2와 YOLOv5를 활용하여 눈의 움직임을 추적하는 인공지능 모델을 구현합니다. 머신 러닝 및 컴퓨터 비전 기술을 결합하여 정확한 눈 움직임 추적 기능을 제공합니다.
